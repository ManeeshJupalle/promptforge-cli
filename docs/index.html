$ cd my-llm-project

Write tests that read like specs

Six assertion types. One YAML file. Every test runs against every provider you list.

prompts/triage.test.yaml YAML 
prompt: ./triage.md
providers:
  - anthropic/claude-sonnet-4-6
  - openai/gpt-4o-mini
  - ollama/llama3.2

tests:
  - name: classifies billing complaint
    vars:
      message: "I was charged twice"
    assert:
      - type: jsonSchema
        schema:
          type: object
          required: [category, urgency]
      - type: contains
        value: billing
      - type: semanticSimilarity
        expected: "I'll escalate for refund"
        threshold: 0.75
      - type: llmJudge
        criteria: "Empathetic + actionable"
      - type: cost
        max: 0.002
      - type: latency
        maxMs: 3000

90 seconds to your first test

No cloud signup, no SDK wiring. Just the four commands. 

# 1. install globally (or as a dev dep)
$ npm install -g promptforge

# 2. scaffold your project
$ cd your-llm-project
$ promptforge init

# 3. run the example that init created (zero API keys needed)
$ promptforge run
